# Android TWA (local Cloudflare)

Bubblewrap project for packaging the Raintech HRM PWA as a Trusted Web Activity.

See **[docs/ANDROID_TWA.md](../docs/ANDROID_TWA.md)** for the full local tunnel loop.

```powershell
.\android-twa\setup-tools.ps1   # once
.\android-twa\build-cloudflare.ps1 -TunnelUrl https://xxxx.trycloudflare.com
```
